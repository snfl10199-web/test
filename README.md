## 게임: 벽돌깨기 (간단)

간단한 HTML5 Canvas 기반의 벽돌깨기 게임 파일을 추가했습니다:

- `index.html` — 게임 실행 페이지
- `style.css` — 간단한 스타일
- `script.js` — 게임 로직 (볼, 패들, 벽돌, 충돌, 점수, 생명)

로컬에서 실행하는 방법:

1. 파일을 브라우저에서 열기 (간단): `index.html` 파일을 더블클릭하거나 브라우저에서 열면 됩니다.

2. 간단한 로컬 서버 (권장, 일부 브라우저에서 로컬 파일 제한이 있을 경우):

PowerShell에서 (현재 디렉터리가 프로젝트 루트일 때):
```
python -m http.server 8000
```

그런 다음 브라우저에서 `http://localhost:8000` 을 열고 `index.html` 을 클릭하세요.

원하시면 다음 작업도 도와드릴게요:
- 키/마우스 조작 튜닝, 난이도 설정, 레벨 추가
- 점수 저장(로컬스토리지), 모바일 터치 지원
